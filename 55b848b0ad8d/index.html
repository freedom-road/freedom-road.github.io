Github + Hexo 免费搭建私人博客网站

Hexo简介

关于Github就不多说了，这里介绍下Hexo。Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。Hexo是基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Coding上，是搭建博客的首选框架。

博客搭建步骤

  1. 安装Git
  2. 安装Node.js
  3. 安装Hexo
  4. GitHub创建个人仓库
  5. 将Hexo部署到GitHub
  6. 设置个人域名 (可选)
  7. 更改主题 icarus
  8. 发布文章

安装Hexo

所有必备的应用程序（Git、Node.js）安装完成后，即可使用 npm 安装 Hexo。

输入 hexo -v 查看一下版本，检查是否安装成功。

初始化Hexo

博客初始化后会生成多个文件夹，每个文件夹的作用如下：

  • node_modules: 依赖包
  • public：存放生成的页面
  • scaffolds：生成文章的一些模板
  • source：用来存放你的文章
  • themes：主题
  • _config.yml: 博客的配置文件
